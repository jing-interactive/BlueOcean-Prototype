# プロトタイプ版 Blue Ocean

![スクリーンショット](screen_shot.png)

新作iOSアプリのプロトタイプ版

## How to build
+ Cinder0.9.0が必要です。
+ Windows版はビルドしていないのでよくわかりません


## Attention
+ iOSはOpenGL ES3.0で動いています。プリプロセッサにCINDER_GL_ES_3を追加してCinderを再ビルドしてください。
